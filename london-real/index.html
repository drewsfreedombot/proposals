Two reasons: (1) If Charlie starts on cold traffic day one, despite training, he will burn some leads. That looks bad — and you'd rightfully question the investment. "Who will I meet?"). This builds Charlie's knowledge base and objection handling in a live environment — so when he hits cold leads, he's already battle-tested. Low risk, high reward.

Configuration Details

How We'll Configure Charlie

Channel architecture, qualification logic, and global setup

🌍 Global Channel Architecture

  • North America: SMS/iMessage via Link. A2P built in. iMessage for first 50 leads/day (blue bubble = higher trust). $500 one-time setup + $250/mo per line.
  • International: WhatsApp via existing Meta WhatsApp account. No additional fees expected. Covers UK, Europe, Dubai, LATAM, ROW.
  • Double-tap strategy: Charlie reaches out on both SMS and WhatsApp. Asks which channel the prospect prefers. Shuts off the one they don't use. Sticks to their preferred channel from there.
  • Ad campaign segmentation: Segment by geography at the ad level so Charlie knows which number/channel to use. UK number for Europe/international. US number for North America.
  • Anti-spam compliance: Link prevents mass blasts. Charlie is persistent but professional. No flagging risk when the talk track is conversational and goal-oriented (not spammy).

💰 Conversational Qualification ($20K Liquidity)

  • Soft qualification: "Curious — what do you do for work?" If they're a CTO/banker/investor → fast-track to call. If signals are ambiguous → ask more.
  • Direct qualification: "Just so you know, the program is $10,000 to join plus a $10,000 minimum investment. Is that palatable?" Filters before burning closer time.
  • Job title data: Can upload global job data to Charlie so he can assess qualification based on stated occupation.
  • DQ handling: If someone clearly can't afford it, Charlie lets them know respectfully and moves them to dead/DQ in ActiveCampaign.

⚙️ Technical Integration

  • CRM: ActiveCampaign — Charlie moves deals between pipeline stages, updates contact statuses, triggers different agents based on status changes
  • Phone: Link (recommended) for SMS/iMessage. Existing Meta WhatsApp for international.
  • Calendar: Confirm booking system on kickoff — Charlie sends invites, books conversationally, or sends calendar links
  • Dialer.io: Charlie is building a dialer.io integration for voice (3–6 month timeline). Drew knows Matt well. Future opportunity for voice outreach integration.
  • Data persistence: Charlie pushes conversation data into ActiveCampaign custom fields (goals, pain points, qualification info). Pulls it back for personalized reactivation follow-ups.

🔇 The Firewall — Isolating Cold Traffic

  • JB's plan: cold leads from new Meta campaigns go exclusively to Charlie — firewalled from the usual email chains and setter outreach
  • Clean A/B test of Charlie's performance on cold traffic without overlapping touchpoints
  • Existing warm funnel stays untouched — Charlie only handles cold leads + dead/no-show recovery from existing funnel
  • Once Charlie proves out on cold, can expand to warm traffic augmentation
